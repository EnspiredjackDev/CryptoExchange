âš™ï¸ Setup Steps

Step 1: Set Admin Access Key

In your main exchange directory, edit the security_config.py file:

ADMIN_ACCESS_KEY = "your-secret-admin-key-here"

This key controls who can access the admin interface at all.

Step 2: Get Your API Key

You need an API key from the main exchange to perform admin operations:

  1. Create a user account on the main exchange
  2. Generate an API key from your account settings
  3. Make sure this API key has admin privileges

Step 3: Access Admin Panel

  1. Go to /admin (this page is hidden from navigation)
  2. Enter your Admin Access Key
  3. Enter your API Key for backend operations
  4. You're now ready to manage the exchange!
